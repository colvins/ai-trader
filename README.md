# AI Trader（日本株 自動選別・分析システム）

日本株を対象にした、テクニカル分析・イベント分析・簡易バックテストを組み合わせた  
軽量な自動銘柄スクリーニングシステムです。

---

## 🚀 概要

本プロジェクトは以下を目的としています：

- 短期的な反発（dip）・トレンド・ブレイクアウトの検出
- TDnet（適時開示）などのイベント情報の反映
- シンプルなバックテストによる戦略検証
- オプション戦略（CALL中心）の補助判断

---

## 🧠 特徴

### 1. テクニカル + イベントの組み合わせ
- 価格・モメンタム・出来高などのテクニカル指標
- TDnet（決算・自社株買いなど）のイベント情報

---

### 2. 軽量バックテスト機能
- 1日 / 3日 / 5日リターン
- 勝率（winrate）
- シグナル別統計

---

### 3. オプション視点の分析
各銘柄に対して：

- CALL（短期反発想定）
- WATCH（様子見）

などの簡易判断を出力

---

### 4. Web可視化
- ローカルK線チャート
- シグナル表示
- 過去パフォーマンス確認

---

### 5. Telegram Bot対応
- 当日シグナル確認
- 戦略モード切替
- バックテスト結果表示

---
